# Security Policy

## Supported Versions
Security vulnerabilities in any version of OptiDesk that is in use in any branch of this or the remote repository are covered by this policy.

## Reporting a Vulnerability

Vulnerabilities should be immediately reported to
> liam@optidesk.dev - Lead Developer

and if no response, to:
> ict@optidesk.dev - Information Technologies Inbox (Urgent)

or, any alternate method that you have that is private.

Your message should include details about what the vulnerability is, the risk and the urgency. We aim to respond within 24-48 hours. The vulnerability will be actioned, and, where indebted, a bug bounty may be awarded at the discretion of OptiDesk.
